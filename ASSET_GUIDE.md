# Asset Guide

## Important folders

- `assets/images/optimized/` — optimized product images
- `assets/images/branded/` — factory / QC images from previous versions
- `assets/images/generated/` — generated or substituted assets for missing materials

## Generated / substituted materials in V11

- `building.webp` — company profile building image
- `factory-ai.webp` — production / factory image
- `rd-ai.webp` and `rd-lab.webp` — R&D / engineering images
- `warehouse-ai.webp` — warehouse / logistics image
- `exhibition-1.webp` to `exhibition-5.webp` — exhibition highlights

## Replace later with official assets

- Real building photo
- Real exhibition photos
- Certificate scans
- Additional product model images
- Project case photos
