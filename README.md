## Deploy to GitHub Pages

1. Unzip the package.
2. Upload the contents of `bolvra-b2b-site-v11` to the root of a GitHub repository.
3. Go to `Settings -> Pages`.
4. Select the main branch and root directory.
5. Save and wait for the website to publish.
